## Getting Started

First, run the development server:

```bash
rm -rf yarn.lock package-lock.json node_modules
npm install --force
npm audit fix --force
npm fund

npm run dev

# or

yarn dev
```






Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
